# Voucher Automation System User Guide

## Introduction
The Voucher Automation System is a specialized tool designed for Nagarkot Forwarders to streamline the processing of General and Job-related vouchers. It automates expense code suggestions, cost center mapping, and generates structured Excel/CSV files compatible with Logisys.

## How to Use

### 1. Launching the App
Simply double-click the `VoucherProcessor.exe` file. The application is portable and remembers your settings and processing history in its local folder.

### 2. The Workflow (Step-by-Step)
1. **Choose Workflow**: Select either **General Vouchers** or **Job Vouchers**.
2. **Upload Files**: Click 'Upload Files' in the sidebar and select the exported Zoho excel file(s).
3. **Verify Settings**: Adjust the **Fiscal Year** and **Default CR/DR** values if necessary.
4. **Assign Codes**: 
   - Review each transaction row. 
   - The system will automatically suggest an 'Expense Code' based on the description.
   - Use the **Master Purpose Code** search at the top-left to apply codes to specific rows.
5. **Copy Down Feature (↓)**:
   - **Single Click**: Copies the current row's value to the row immediately below.
   - **Multiple Clicks**: Clicking the same button again will push the value further down (3rd row, 4th row, etc.).
   - **Automatic Reset**: Manual changes to any dropdown or using the Master Purpose Code will reset the copy sequence to ensure accuracy.
6. **Generate Output**: Click 'Generate Logisys Excel'. 
   - Choose a target folder to save your files.
   - If duplicates are detected (records already processed), the system will ask if you want to overwrite or skip them.

## Interface Reference

| Control / Input | Description | Expected Format |
| :--- | :--- | :--- |
| **Workflow Cards** | Selects the processing mode: General or Job. | Interaction |
| **Upload Files** | Opens a file dialog to select input Excel files. | `.xlsx` files |
| **Fiscal Year** | Sets the year used in the generated Job Numbers. | e.g., 25-26 |
| **Master Purpose Code** | A searchable dropdown to bulk-apply a code to rows. | Search/Click |
| **↓ Button** | Sequential copy-down button (Click multiple times to fill further). | Action Button |
| **LOB/Branch** | Automatically determined; adjusts Cost Center and Job No. | Dropdown Selection |

## Troubleshooting & Validations

If you see an error, check this table:

| Message | What it means | Solution |
| :--- | :--- | :--- |
| `[LOGO MISSING]` | The branding image failed to load internally. | Ensure you are running the official bundled version. |
| `Validation Failed: X Codes missing` | One or more rows do not have an Expense Code selected. | Ensure every row has a code from the searchable dropdown. |
| `Duplicate Output Files Detected` | You have already processed these Transaction IDs before. | Choose **[No]** (Skip) to only process new/unique records. |
| `Failed generating output` | Usually occurs if the input file format has changed. | Verify your input Excel has the required 'Transaction ID' and sheet names. |
